Algumas outras formatações

Código-fonte e Pré formatação

Existem algumas tags para conseguirmos representar códigos de forma mais verídica
Essas tags são muito úteis na hora de apresenta-los:
São a tag <code> e tag <pre>, onde a primeira vai igualar o tamanho das letras e a segunda vai demonstrar a tabulação, observe:

Veja o que ocorre se representarmos sem usar as tags:
num1 = float(input("Digite o primeiro número: ")) num2 = float(input("Digite o segundo número: ")) soma = num1 + num2 subtracao = num1 - num2 multiplicacao = num1 * num2 divisao = num1 / num2
Além de não organizar, vai ficar em uma fonte de difícil compreensão

Agora observe com a tag <code>:
num1 = float(input("Digite o primeiro número: ")) num2 = float(input("Digite o segundo número: ")) soma = num1 + num2 subtracao = num1 - num2 multiplicacao = num1 * num2 divisao = num1 / num2
Ocorre a mudança para uma fonte mais fácil, porém sem organizar

Para organizar, o que faz mais sentido é usar a tag <pre>, onde ela vai pegar a tabulação(espaçamentos) da página: 

                    
                num1 = float(input("Digite o primeiro número: "))
                num2 = float(input("Digite o segundo número: "))
                
                soma = num1 + num2
                subtracao = num1 - num2
                multiplicacao = num1 * num2
                divisao = num1 / num2

O que é tabulação e quais são os atalhos?

A tabulação é o espaçamento feito pelo próprio programa
Existem alguns atalhos para agilizar o trabalho, sendo eles:
Shift + Tab - para mover para a esquerda
Tab - para mover o código para a direita

Agora, vamos ver um pouco mais sobre: CITAÇÕES

Existem as simples e as completas, com as tags:
Simples <q>: Observe o que ele me disse: EU TE ODEIO.
Completas <Blockquote cite = "URL da fonte">: Já disse Albert Einstein:

Faça as coisas o mais simples que você puder, porém não se restrinja às mais simples.

Essas tags agregam significado ao texto, então é melhor do que simplesmente escrever as aspas

Abreviações/Inversões

Quando escrevemos siglas/abreviações, existe a tag <abbr>, para que coloquemos um pop-up explicando o que é aquilo.

Também é possível inverter um texto dependendo da sua direção, usando a tag <bdo: direction>, como o exemplo a seguir:
Eu estava andando sozinho. - Esse texto estava da direita p/esquerda
