Vídeos locais

Bom, até agora já vimos como imagens dinâmicas e áudios funcionam.
Agora, vamos ver como funcionam os vídeos. Basta utilizar a tag <video>. 🤓
A partir de agora vamos usar somente a versão complexa,
sem levar em consideração a recomendação do VScode.

Lembre-se das tags controls(para tocar o vídeo), width(muda o tamanho do vídeo) e poster(para as thumbs).
É possível usar o CSS para mudar o tamanho do vídeo também.
Lembre-se que colocar vídeos de maneira local pode comprometer o preço da sua hospedagem.
