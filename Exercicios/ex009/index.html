Listas de definição/Definition Lists

Listas de definição funcionam com três tags:

  • <dl> (definition list)
  • <dt> (definition term)
  • <dd> (definition description)

Observe a seguir:

HTML5
Linguagem de marcação de hipertexto, usada com característica semântica
CSS
Linguagem de estilo com cascata
Javascript
Linguagem de programação client/server-side
