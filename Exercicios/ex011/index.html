Imagens dinâmicas

Quando fazemos imagens dinâmicas, precisamos de duas tags importantes,
a tag <picture>(que vai englobar tudo)
E a tag <source>, que permite diversas imagens em cima de uma base 🤓

Observe:

Lembre-se de pegar sempre os pixels
300px pegam certinho o tamanho do celular
Veja aqui minhas redes para aprender mais

Áudios

Colocar áudios é uma tarefa bem simples, podemos simplesmente usar a tag <audio>,
Tanto de forma Simples quanto Complexa
Observe:

Lembrando que dessa forma o site está baixando o arquivo completamente antes mesmo de tocá-lo
Para mudar isso, ou colocar diferente tipos de arquivo como WAV, MP3 ou OGG.
Basta colocar a tag <audio> como colocamos a tag <picture>.
Observe:

O preload como metadata vai carregar o áudio enquanto você ouve, lembre-se de tirar o autoplay nesses casos.
Funciona bem com arquivos grandes.

Infelizmente seu navegador não consegue reproduzir o arquivo.
